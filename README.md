# VisualSilence / Hypr-Lab

Hello! I'm Ádám, a Linux enthusiast from Hungary.

I've been using Linux for about six months and currently run a vanilla Arch Linux installation as my daily operating system.

At the moment, I'm working on my personal project called **Hypr-Lab**, which is still in its early stages. Everything in this repository is being built through self-learning, using documentation, wikis, GitHub repositories, YouTube tutorials, and AI-assisted guidance.

## What is VisualSilence?

**VisualSilence** represents my personal philosophy of how a desktop environment should look and feel.

Productivity is important to me. I want my desktop to make efficient use of available screen space and support my workflow without unnecessary distractions. At the same time, appearance matters too. A desktop should be visually pleasing, but it should never compete with the work being done on it.

My goal is to find a balance between aesthetics, simplicity, and functionality.

## What is Hypr-Lab?

**Hypr-Lab** is the codename of this project.

I chose this name because I started my Hyprland journey as a complete beginner. This repository is my playground, workshop, and learning environment.

Instead of modifying a fully polished desktop configuration created by someone else, I decided to build my own from the ground up. Here I experiment, test ideas, make mistakes, learn new things, and slowly shape a desktop environment that fits my personal workflow.

In many ways, Hypr-Lab is less about the final result and more about the learning process itself.

## Repository Status

The VisualSilence repository is currently private and serves as both a personal backup and project documentation archive.

Once I feel the project has reached a presentable and stable state, the repository will become public and available to everyone.
